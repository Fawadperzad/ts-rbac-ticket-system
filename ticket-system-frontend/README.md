# TicketSystemFrontend

A modern frontend application for a Ticket Management System built with **Angular 21**, **TypeScript**, and **Angular CLI**.

---

## 🚀 Overview

TicketSystemFrontend is the user interface of a helpdesk and ticket management system. It provides an intuitive dashboard for managing support tickets, users, and authentication.

### Features

* User Authentication
* Dashboard
* Ticket Management
* Responsive UI
* REST API Integration
* Angular Routing
* Modular Architecture

---

## 🛠️ Technologies

* Angular 21
* TypeScript

* CSS3

## 📋 Prerequisites

Before running the project, install:

* Node.js (Latest LTS)
* npm
* Angular CLI

Install Angular CLI globally:

```bash
npm install -g @angular/cli
```

---

## 📦 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd TicketSystemFrontend
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Development Server

Start the local development server:

```bash
ng serve
```

Open your browser:

```
http://localhost:4200
```

The application automatically reloads when source files are modified.

---

## 🏗️ Build

Create a production build:

```bash
ng build
```

The compiled files will be generated inside the:

```
dist/
```

directory.

## 🧩 Generate Components

## ✅ Unit Testing

Run unit tests using Vitest:

```bash
ng test
```

---

## 🔍 End-to-End Testing

Run end-to-end tests:

```bash
ng e2e
```

Angular CLI does not include an e2e framework by default. You may choose your preferred framework such as Cypress or Playwright.
